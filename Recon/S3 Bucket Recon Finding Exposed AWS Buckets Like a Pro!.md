- **use #tool/dork-eye from dir tools give it the dork and let it give you urls**
- **after this use tool #tool/s3bucketmisconfig and pass the file generated by dork eye that tool will validate all the buckets**

manual valid bucket check:
```
aws s3 ls s3://{bucket name} --no-sign-request
```

httpx command to find s3:
```
subfinder -d somaiya.edu -all -silent | httpx-toolkit -sc -title -td | grep "Amazon S3"
```
nuclei:
```
subfinder -d nasa.gov -all -silent | nuclei -t nuclei-templates/http/technologies/s3-detect.yaml
```

download files to check for read access:
```
aws s3 cp s3://svu-admissions/Curriculum.pdf ./ --no-sign-request
```

katana to extract s3 buckets from js files:
```
katana -u https://nasa.gov/ -d 5 -jc | grep '\.js$' | tee alljs.txt  
cat alljs.txt | xargs -I {} curl -s {} | grep -oE 'http[s]?://[^"]*\.s3\.amazonaws\.com[^" ]*' | sort -u
```

OR

use #tool/java2s3:
1) first get your live subdomains:
2) remove the protocol:
3) run the tool:
```
subfinder -d nasa.gov -all | httpx -o file.txt
cat file.txt | grep -oP '(?<=https?:\/\/).*' >input.txt
python java2s3.py input.txt nasa.gov output.txt
```
4) Find juice:
```
cat output.txt | grep -E "S3 Buckets: \['[^]]+"  
cat output.txt | grep -oP 'https?://[a-zA-Z0-9.-]*s3(\.dualstack)?\.ap-[a-z0-9-]+\.amazonaws\.com/[^\s"<>]+' | sort -u  
cat output.txt | grep -oP '([a-zA-Z0-9.-]+\.s3(\.dualstack)?\.[a-z0-9-]+\.amazonaws\.com)' | sort -u
```
 after getting all s3 url from here you can feed all url to #tool/s3bucketmisconfig to validate the findings

also can use #tools/lazys3 to brute force for s3 buckets:
```
ruby lazys3.rb nasa.gov
```

OR 

you can generate a custom wordlist on the basis of the site using #tools/cewl
and then pass the s3 urls to #tool/s3scanner 
```
cewl https://site.com/ -d 3 -w file.txt  
  
s3scanner -bucket-file file.txt -enumerate -threads 10 | grep -aE 'AllUsers: \[.*(READ|WRITE|FULL).*]'
```

#AWS S3 Bucket Listing & File Management:
Read File: aws s3 ls s3://[bucketname] --no-sign-request
Copy Files: aws s3 cp xss.svg s3://[bucketname] --no-sign-request
Delete Files: aws s3 rm s3://[bucketname]/xss.svg --no-sign-request
Owner Check: aws s3api get-bucket-acl --bucket [bucketname]
Download Files: aws s3 cp s3://[bucketname]/somefile.txt ./ --no-sign-request
Download all Files & Folders: aws s3 cp s3://[bucketname]/ ./download --recursive --no-sign-request
