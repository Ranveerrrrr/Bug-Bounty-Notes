HTML GET Request Examples

This page shows different ways an HTML file can trigger GET requests.

1. GET via link

Clicking this link sends a GET request:

Send GET request

GET /?test=link

2. GET via form



GET //etc/natas_webpass/natas13

3. Automatic GET (image)

This request is sent automatically when the page loads:

GET /?auto=1

4. GET via JavaScript (fetch)



    

    
GET /?js=fetch

  


  


  

    💡 Tip for CTFs:  
    You can usually do all of this just by editing the URL in your browser’s
    address bar or by modifying parameters and cookies in DevTools.
